# Smart Tracker Docs

## Overview

Smart Tracker is:

1. A modern, flexible workflow management system
2. An operational intelligence engine
3. An integration layer across external systems

It helps teams understand:

* what matters
* why it matters
* what to do next

## Philosophy

Smart Tracker is not a task manager.

It is a **decision engine layered on top of operational work**.
