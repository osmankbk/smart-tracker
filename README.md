# Smart Tracker

Smart Tracker is a full-stack monorepo project for tracking orders, tasks, and items through their lifecycle, with future support for AI-generated summaries and operational insights.

## Stack

- Next.js
- NestJS
- PostgreSQL
- Redis
- Docker
- AWS (planned)

## Apps

- `apps/web` — frontend
- `apps/api` — backend

## Run locally

### Install

```bash
npm install
```
